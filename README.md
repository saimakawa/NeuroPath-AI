# FutureLens AI

FutureLens AI is an AI-powered web application that tracks skill decay using time-based half-life modeling and predicts the longevity of startup ideas.

## Features
- User Authentication
- Skill Half-Life Tracker
- Idea Longevity Predictor
- Smart Skill Recommendation
- Dashboard Visualization

## Tech Stack
Frontend: React.js  
Backend: Node.js, Express.js  
Database: MongoDB  
AI: Python  

## Installation

1. Clone repository
2. Install backend dependencies
   npm install
3. Install frontend dependencies
   cd client
   npm install
4. Run backend
   npm start
5. Run frontend
   npm run dev

## Usage
Register → Add Skills → Analyze Idea → View Dashboard
